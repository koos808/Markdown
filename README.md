markdown
===

* Frequently used mark down code
* 내가 자주 사용하는 마크다운 문법

0.제목
---
```
#  h1
##
...
###### h6

h1에서 h6으로 갈수록 글씨 크기 작아짐.
```

1.헤더(Header)
----
```
헤더 밑에 === 입력
```


2.스몰 헤더(Small Header)
----
```
스몰헤더 밑에 ---입력
```


*3.글씨체, 강조 : Emphasis*
---
~~~
1. bold(굵게 쓰기) : 양쪽에 __ 입력
2. italic(기울기)  : 양쪽에 * or _ 입력
~~~

4.코드삽입
---
```
* 펜스 코드 블럭 지정하고 싶을땐
양쪽에 ``` or ~~~ 지정

* 한줄에 코드 지정하고 싶을 떈
양쪽에 ` 지정 or space bar 4개
```

5.공백삽입, 행 여러줄 띄울 때
---
```
<br /> 입력하면 한 줄 공백생김.
```
